# Start Learning

Begin your developer roadmap journey by choosing your learning path.

## Usage

`/start-learning`

## What This Does

When you run this command, Claude will help you:

1. **Understand Available Paths** - Discover all 7 developer roles
2. **Assess Your Background** - Understand where you're starting from
3. **Recommend a Path** - Suggest the best role based on your interests
4. **Create a Plan** - Design your personalized learning roadmap
5. **Set Goals** - Define milestones and timelines

## Available Roles

- **Frontend Developer** - Build beautiful user interfaces
- **Backend Developer** - Create powerful server systems
- **DevOps Engineer** - Manage infrastructure and deployments
- **Data Scientist** - Extract insights from data
- **Full Stack Developer** - Master both frontend and backend
- **Cloud Engineer** - Design cloud architectures
- **AI Engineer** - Build intelligent systems with LLMs

## What to Expect

After running this command, you'll get:
- Overview of each role
- Time estimates for different paces
- Required prerequisites
- First steps to begin
- Resource recommendations

## Example

```
User: /start-learning
Claude: Let's find your perfect role! First, tell me:
- What's your programming experience level?
- What excites you most about development?
- How much time can you dedicate per week?
```

## Next Commands

Once you choose a role, try:
- `/roadmap-status` - Check your progress
- `/skill-deep-dive` - Learn specific skills
- `/project-ideas` - Get hands-on projects
