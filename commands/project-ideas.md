# Project Ideas

Get tailored project ideas based on your current skill level and learning path.

## Usage

`/project-ideas [level] [role]`

## Levels

- **Beginner** - First projects to learn fundamentals
- **Intermediate** - Build real-world applications
- **Advanced** - Complex features and architecture

## What You'll Get

- Complete project specifications
- Technology stack recommendations
- Feature breakdown
- Implementation tips
- Learning outcomes
- Estimated time (hours/weeks)
- Success criteria
- Portfolio value

## Examples

```
/project-ideas beginner frontend
→ Todo App, Portfolio Website, Weather App

/project-ideas intermediate backend
→ Blog Platform, E-commerce API, Real-time Chat

/project-ideas advanced fullstack
→ SaaS Platform, AI Assistant, Analytics Dashboard
```

## How to Use These Projects

1. **Read the spec** carefully
2. **Plan your architecture** before coding
3. **Build incrementally** (MVP first)
4. **Add testing** as you go
5. **Deploy to production**
6. **Publish on GitHub** with great documentation
7. **Add to your portfolio**

## After Each Project

- Deploy and share the live link
- Write a blog post about what you learned
- Open source it on GitHub
- Get feedback from community
- Iterate and improve
